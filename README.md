# dotfiles

A personal collection of dotfiles and scripts for syncing settings across machines.

## Setup

Run the following commands to set up dotfile links:

```bash
./install.sh
```

## Update Things

Using the `update.sh` script, you can update installed modules:

```bash
./update.sh
```

or

```bash
bun run update
```
